Описание задания
Реализуйте многопоточную систему кэширования или событий, используя различные потокобезопасные коллекции и шаблоны из java.util.concurrent. Ваша реализация может быть основой для таких систем, как:

система уведомлений,

логгирование событий,

обработка событий пользователей в веб-приложениях.

Составьте отчет, в котором кратко опишите результаты выполнения задания.

Ссылку на отчет прикрепите в личном кабинете на платформе.

 
Чек-лист самопроверки:

Реализована многопоточная система кэширования или событий с помощью потокобезопасных коллекций и шаблонов из java.util.concurrent.

В ЛМС прикреплена ссылка на отчёт, в котором описаны этапы выполнения задания.
