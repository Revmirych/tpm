Цель задания

Научиться работать с различными типами пулов потоков (FixedThreadPool, CachedThreadPool, SingleThreadExecutor).
Получить навыки настройки пула потоков, управления задачами и мониторинга состояния пула.

Описание задания
Реализуйте приложение, которое:

принимает список URL-адресов (например, 20 публичных API и т.д.),

отправляет асинхронные HTTP-запросы ко всем этим URL с использованием собственного ThreadPoolExecutor, настроенного вручную,

собирает результаты (статус-коды и время ответа) и выводит сводку.

 
Составьте краткий отчёт с описанием каждого этапа задания.

Ссылку на отчёт прикрепите в личный кабинет в ЛМС.
