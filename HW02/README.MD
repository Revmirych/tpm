Описание задания
Напишите программу, которая демонстрирует различные состояния потока (NEW, RUNNABLE, WAITING, BLOCKED, TIMED_WAITING, TERMINATED).
Важно: программа должна содержать, как минимум, 3 потока, каждый из которых будет проходить через разные состояния.

Реализуйте логику, которая позволяет наблюдать за изменением состояний этих потоков в консоли.

Оформление отчёта:Составьте короткий отчет, в котором кратко опишите каждый этап в
Ссылку на отчет прикрепите в личном кабинете на платформе.

Чек-лист самопроверки:
– написана программа, демонстрирующая различные состояния потока;

– программа содержит не менее трёх потоков

– каждый из трёх потоков проходит через разные состояния;

– изменение состояний потоков отображается в консоли.
