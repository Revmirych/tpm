Описание задания

Продумайте архитектуру итогового приложения, создать каркас проекта на Spring Boot по своим вариантам.

Создайте простой класс сервиса. Создайте класс задачи и научитесь создавать и запускать потоки через Thread и Runnable. Оба потока должны выводить на экран имя потока и порядковый номер.

Назовите потоки понятными именами (например, “CounterWorker”, “LoggerThread”)

Выведите список всех активных потоков.

Напишите краткий отчёт, в котором кратко приведите описание каждого этапа выполнения задания, ссылку на отчёт приложите в ЛМС.

Чек-лист самопроверки:
В отчёте описана:

архитектура итогового приложения;
создан простой класс сервиса и класс задачи, созданы и выведены потоки с присвоенным именем и номером.
